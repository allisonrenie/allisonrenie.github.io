Projects

Forest builder

Programmed a basic forest creation program in OpenGL. Allows user to place randomly generated trees on a plane, then view those trees from an overhead view or a walk-through view. This was a class project for CS450 (Intro to Computer Graphics).

Moss shader

AKA the Mossifier. Programmed a moss shader using GLSL and OpenGL. Esentially puts a moss-like texture onto any OBJ file loaded into the program. There are a few options to the moss shader, inlcuding a few moss patterns generated using noise. The background textures have normal mapping. This was a class project for CS457 (Computer Graphics Shaders).

This one is a little goofy, but I made this entire animation in Blender, including creating all the models. This was a class project for CS499: Computer Animation.
