Honors thesis: "Interactive Visualization of Douglas Fir Forest Growth Using 3-PG"

My project combines a complex numerical simulation with a user interface and 3D graphics to create an interactive forest visualization. I created a new implementation of the existing and well-trusted forest growth model 3-PG in C/C++ and added graphics with OpenGL and a user interface using GLUT. While 3-PG has been added to in many ways over the years since its creation in 1997 (by Dr. Landsberg and Dr. Waring), there has never been a 3-D visualization for it, and very little in the way of a user interface (most implemtations are in Excel, with a few in R and a new one in FORTRAN). Basically, you can control and change various input parameters to the simulation and see how they affect the forest in real time. The idea behind this was to add a new layer of discovery and interactivity to the model. I worked with my mentor Dr. Mike Bailey on this project from the spring of 2022 until this May.

Skills: C/C++, OpenGL, GLSL, GLUT, user interface creation, creating a simulation, programming and organizing complex equations.

Capstone senior group project: Rigid body dynamics driving simulation in Bevy using Rust

The idea behind this project was to create an engineering simulation using a game engine to see how well this would work for simulation creators (as opposed to making their own simulations from scratch). We used the game engine Bevy (which is Open Source and Rust based) to create a driving simulation with rigid body dynamics algorithms. My focus in the project was on the physics and how to simulate the different parts of the car. I learned a lot from this project about simulating a complex machine. This project also gave me the chance to become more familiar with Rust.

Skills: Rust, Bevy game engine, physics (RBDA), creating a simulation, team work and communication.

Moss shader

AKA the Mossifier. Programmed a moss shader using GLSL and OpenGL. Esentially, it puts a moss-like texture onto any OBJ file loaded into the program. There are a few options to the moss shader, inlcuding a few moss patterns generated using noise. The background textures have normal mapping. This was a class project for CS457 (Computer Graphics Shaders).

Skills: Creating shaders with GLSL, working with noise, OpenGL.
